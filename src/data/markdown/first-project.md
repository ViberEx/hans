# 创建第一个项目

让我们通过一个实际案例，完整体验如何使用 Lovable 创建一个功能完整的 Web 应用。

## 项目目标

我们将创建一个 **任务管理应用**，功能包括：

- ✅ 添加、编辑、删除任务
- 🏷️ 任务分类（工作、个人、学习）
- ✨ 现代化的 UI 设计
- 📱 响应式布局

## 步骤 1: 初始化项目

在对话框中输入：

```
创建一个任务管理应用的首页，包含：
1. 顶部导航栏，显示"我的任务"标题
2. 添加任务的输入框和按钮
3. 任务列表区域
4. 使用现代风格的设计，主色调为蓝色
```

等待 Lovable 生成基础框架。

## 步骤 2: 完善功能

查看生成的页面后，继续优化：

```
添加以下功能：
- 每个任务显示完成状态的复选框
- 任务右侧显示删除按钮
- 已完成的任务显示删除线效果
- 添加动画效果
```

## 步骤 3: 添加分类

让应用更实用：

```
添加任务分类功能：
- 在添加任务时可以选择分类（工作、个人、学习）
- 不同分类使用不同的颜色标签
- 添加筛选按钮，可以按分类查看任务
```

## 步骤 4: 优化样式

使用可视化编辑进行细节调整：

1. 点击聊天框左侧的 "Visual Edit" 按钮
2. 直接点击页面上的元素
3. 修改颜色、字体大小、间距等
4. 点击 "Save" 保存更改

> 💡 提示: 可视化编辑不消耗积分，适合快速调整样式！

## 步骤 5: 添加数据持久化

如果想让数据保存到数据库：

```
我想让任务数据保存到数据库，刷新页面后不会丢失
```

Lovable 会提示您启用 Lovable Cloud，点击确认即可。

## 完成！

您已经创建了第一个功能完整的应用！接下来可以：

### 测试功能
在右侧预览窗口中测试所有功能是否正常工作。

### 部署应用
1. 点击右上角的 "Publish" 按钮
2. 应用会自动部署
3. 获得一个公开的网址，可以分享给其他人

### 连接 GitHub
1. 点击 "GitHub" 按钮
2. 连接您的 GitHub 账号
3. 代码会自动同步到仓库
4. 支持双向同步
