# 什么是 Lovable

Lovable 是一个革命性的 AI 驱动的全栈 Web 应用开发平台，让您可以通过自然语言对话来构建专业的 Web 应用。

## 核心能力

### 🤖 AI 驱动开发

Lovable 的 AI 助手能够理解您的需求，并自动生成高质量的代码。您只需用自然语言描述想要的功能，AI 就能：

- 创建完整的页面布局和组件
- 实现复杂的交互逻辑
- 设计响应式界面
- 生成数据库架构
- 配置用户认证系统

### 🎨 可视化编辑

Visual Edits 功能让您可以直接在页面上选择和编辑元素：

- **即时编辑**: 直接修改文本、颜色、字体，无需消耗积分
- **提示词编辑**: 选中元素后，可以用自然语言描述更复杂的修改
- **实时预览**: 所有更改立即显示，满意后再保存
- **静态元素支持**: 可以编辑标题、按钮、背景等静态元素

> 💡 **提示**: Visual Edits 的直接编辑功能完全免费，非常适合快速调整样式！

### 💻 现代技术栈

Lovable 基于业界领先的技术构建：

- **前端框架**: React 18 + TypeScript
- **样式方案**: Tailwind CSS + shadcn/ui
- **构建工具**: Vite（快速的开发服务器和构建）
- **状态管理**: TanStack Query
- **路由**: React Router
- **后端**: Lovable Cloud（无需配置的全栈解决方案）

### 🚀 Lovable Cloud

集成的后端服务，提供完整的后端能力：

- **PostgreSQL 数据库**: 自动生成表结构，无需编写 SQL
- **用户认证**: 支持邮箱、手机号、Google 登录
- **文件存储**: 安全的文件上传和管理
- **Edge Functions**: 无服务器函数，自动扩展
- **Secrets 管理**: 安全存储 API 密钥和敏感信息

### 🔧 开发者友好

- **实时预览**: 左侧对话，右侧即时看到效果
- **代码可见**: 开启 Dev Mode 查看和编辑代码
- **GitHub 集成**: 双向同步代码仓库
- **本地开发**: 支持导出项目到本地
- **版本控制**: 自动保存历史版本，随时回滚

## 积分系统

Lovable 使用积分制度：

- **每日积分**: 所有用户每天获得 5 个免费积分
- **聊天模式**: 每条消息消耗 1 个积分
- **Visual Edits 直接编辑**: 完全免费，不消耗积分
- **付费计划**: 提供更多月度积分

> 📌 **注意**: 免费计划的每日积分每月最多累计 30 个

## 最佳实践

### ✅ 推荐做法

1. **分步骤开发**: 不要一次要求太多功能，先搭建基础框架，再逐步添加功能
2. **利用 Visual Edits**: 对于简单的样式调整，使用 Visual Edits 免费编辑
3. **清晰的描述**: 提供具体的需求描述，包括功能和视觉要求
4. **测试再发布**: 在预览环境充分测试后再点击 Publish

### ❌ 避免做法

1. 一次性要求构建整个复杂应用
2. 使用聊天修改简单的文本和颜色（应该用 Visual Edits）
3. 笼统的需求描述（如"做一个网站"）
4. 跳过测试直接发布
