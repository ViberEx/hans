# 快速开始

本指南将帮助您在 10 分钟内创建并发布您的第一个 Lovable 应用。

## 第一步：创建账号

1. 访问 [lovable.dev](https://lovable.dev)
2. 点击右上角的 **"Sign Up"** 或 **"开始使用"**
3. 选择注册方式：
   - Google 账号
   - GitHub 账号
   - 邮箱注册
4. 完成注册流程

> 💡 推荐使用 Google 或 GitHub 账号，可以快速完成注册并支持后续的集成功能。

## 第二步：创建项目

登录后进入项目仪表板：

1. 点击 **"New Project"**（新建项目）
2. 输入项目名称，例如："我的第一个应用"
3. 点击 **"Create"** 创建
4. 几秒钟后，您将进入项目编辑器

## 第三步：了解界面

项目编辑器分为三个主要区域：

### 左侧 - AI 对话区
- 输入需求的聊天框
- 查看对话历史
- Visual Edit 按钮

### 右侧 - 实时预览区
- 实时显示应用效果
- 支持移动端/桌面端切换
- 可以直接交互测试

### 顶部工具栏
- **Dev Mode**: 查看/编辑代码
- **GitHub**: 连接代码仓库
- **Publish**: 发布到生产环境
- **Settings**: 项目设置

## 第四步：创建第一个页面

在聊天框中输入您的第一个提示：

```
创建一个个人介绍网站首页，包含：
- 顶部导航栏，有"首页"、"关于"、"项目"三个链接
- 一个英雄区块，标题是"欢迎来到我的网站"，副标题是"全栈开发者"
- 使用现代化的蓝色渐变背景
- 添加一个"联系我"的按钮
```

按 Enter 发送，等待 AI 生成...

### 观察变化

10-30 秒后，您会看到：
- 左侧显示 AI 的工作进度
- 右侧预览窗口显示生成的页面
- 页面包含您要求的所有元素

## 第五步：迭代优化

不满意？继续对话进行调整：

### 修改样式
```
把导航栏改成半透明的，背景从蓝色渐变改成紫色到粉色
```

### 调整内容
```
把标题改大一些，字体加粗，添加一个动画效果
```

### 添加功能
```
在页面下方添加一个"关于我"区块，包含三张卡片展示我的技能
```

> 💡 **提示**: Lovable 会记住之前的对话，可以持续迭代优化。

## 第六步：使用 Visual Edits

对于简单的文本和样式修改，使用 Visual Edits 更快且免费：

1. 点击聊天框左下角的 **"Edit"** 按钮
2. 鼠标悬停在页面元素上
3. 点击要编辑的元素（如标题）
4. 直接修改文本、颜色、字体大小
5. 点击 **"Save"** 保存

> 🎉 **免费功能**: Visual Edits 的直接编辑不消耗积分！

## 第七步：查看代码（可选）

想看看 AI 生成了什么代码？

1. 点击左上角的 **"Dev Mode"** 开关
2. 左侧显示文件树
3. 点击文件查看代码
4. 所有代码都是标准的 React + TypeScript

> 📝 **手动编辑**: 需要在设置中启用"Enable Code Editing"才能手动修改代码。

## 第八步：发布应用

满意了？一键发布到互联网！

### 桌面端发布
1. 点击右上角的 **"Publish"** 按钮
2. 等待 30-60 秒
3. 获得公开访问链接：`https://your-project.lovable.app`

### 移动端发布
1. 切换到 Preview 模式
2. 点击右下角的 **"Publish"** 按钮

### 更新已发布的应用
- 修改后再次点击 Publish
- 在弹出对话框中点击 **"Update"**
- 生产环境会更新为最新版本

> ⚠️ **注意**: 前端更改需要手动点击 Update 才会发布，后端更改会自动部署。

## 提示词技巧

### ✅ 好的提示词

**具体明确**
```
创建一个登录表单，包含邮箱输入框、密码输入框和登录按钮，
使用卡片样式，居中显示，添加表单验证
```

**包含细节**
```
添加一个商品卡片，显示图片、标题、价格和"加入购物车"按钮，
卡片有阴影效果，鼠标悬停时有上浮动画
```

**分步进行**
```
第一步：创建一个博客文章列表页面
（测试后）第二步：添加文章详情页
（测试后）第三步：添加评论功能
```

### ❌ 避免的提示词

**过于笼统**
```
做一个网站  ❌
建一个商城  ❌
```

**要求太多**
```
创建一个包含首页、产品列表、详情、购物车、结账、
用户中心、订单管理、评论、收藏等所有功能的电商网站  ❌
```

**缺少描述**
```
做个表单  ❌
加个按钮  ❌
```

## 常见问题

### Q: 每次修改都会消耗积分吗？

A: 聊天模式的每条消息消耗 1 个积分。但 Visual Edits 的直接编辑（文本、颜色、字体）完全免费！

### Q: 可以撤销修改吗？

A: 可以！Lovable 自动保存版本历史，在编辑器中可以查看和回滚到之前的任何版本。

### Q: 生成的代码可以导出吗？

A: 可以！点击 GitHub 按钮连接仓库，代码会自动同步。您也可以下载整个项目到本地。

### Q: 支持移动端吗？

A: 当然！所有生成的应用都是响应式的，自动适配手机、平板和桌面。

### Q: 如何添加数据库功能？

A: 告诉 AI："我需要把数据保存到数据库"，系统会提示您启用 Lovable Cloud。

## 积分使用建议

为了更高效地使用积分：

1. **先规划后实现**: 想清楚要做什么再开始
2. **使用 Visual Edits**: 简单修改用免费的 Visual Edits
3. **分批测试**: 完成一个功能测试后再继续
4. **具体描述**: 清晰的提示词减少来回修改
5. **学习文档**: 了解最佳实践，减少试错
