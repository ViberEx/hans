# 快速开始

本指南将帮助您快速上手 Lovable，在几分钟内创建您的第一个 Web 应用。

## 创建账号

1. 访问 [lovable.dev](https://lovable.dev)
2. 点击 "开始使用" 或 "Sign Up"
3. 使用 Google、GitHub 或邮箱注册账号
4. 完成邮箱验证（如适用）

## 创建项目

登录后，您将看到项目仪表板：

1. 点击 "新建项目" (New Project)
2. 输入项目名称
3. 系统会自动创建一个空白项目

## 开始对话

进入项目后，您会看到：

- **左侧**: AI 对话框，用于输入需求
- **右侧**: 实时预览窗口

### 第一个提示词

在对话框中输入您的第一个需求，例如：

```
创建一个现代风格的个人网站首页，包含：
- 导航栏（首页、关于、项目、联系）
- 英雄区块，标题是"全栈开发者"
- 使用蓝色渐变背景
```

## 实时预览

提交后，AI 会：

1. 理解您的需求
2. 生成相应的代码
3. 在右侧窗口实时显示效果

通常只需 10-30 秒就能看到结果！

## 迭代改进

对结果不满意？继续对话：

```
把导航栏改成透明的，背景渐变改成紫色到蓝色
```

Lovable 会记住之前的对话内容，持续优化您的应用。

## 查看代码

想查看生成的代码？

1. 点击左上角的 "Dev Mode" 切换按钮
2. 在左侧会显示文件树和代码编辑器
3. 您可以查看和手动编辑代码（需要在设置中启用）

## 提示词技巧

### ✅ 好的提示词

- 具体明确："创建一个包含邮箱和密码输入框的登录表单"
- 包含细节："使用卡片样式，添加阴影效果，圆角 8px"
- 逐步迭代：先创建基础功能，再优化细节

### ❌ 避免的提示词

- 过于笼统："做一个网站"
- 一次要求太多功能
- 没有明确的视觉或功能描述
