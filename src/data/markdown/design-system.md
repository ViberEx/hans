# 设计系统

设计系统是保持应用视觉一致性的关键。Lovable 基于现代化的设计原则，使用 Tailwind CSS 和 shadcn/ui 构建美观且统一的界面。

## 什么是设计系统

设计系统定义了应用的视觉语言和交互规范：

- 🎨 **颜色方案** - 主色、辅助色、语义色
- 📏 **间距系统** - 统一的 margin、padding 规范
- ✍️ **字体排版** - 字号、字重、行高层级
- 🔘 **组件样式** - 按钮、卡片、表单等 UI 组件
- 🌗 **主题模式** - 明亮/暗色模式自动切换

## Lovable 的技术栈

### Tailwind CSS

实用优先的 CSS 框架，直接在 HTML 中编写样式。查看完整内容请访问官方文档。
