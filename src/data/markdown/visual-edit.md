# 可视化编辑

可视化编辑是 Lovable 的强大功能，让您可以直接在页面上选择元素并快速修改，无需编写提示词。

## 启用可视化编辑

1. 在聊天框左下角找到 **"Edit"** 按钮
2. 点击激活可视化编辑模式
3. 您的鼠标现在可以选择页面上的元素

## 可以编辑的元素

### ✅ 可选择的元素

- 文本内容（标题、段落、按钮文字等）
- 静态图片
- 背景色块
- 容器和卡片
- 导航栏项目
- 按钮和链接

### ❌ 不可选择的元素

- 动态生成的内容
- 列表中的循环项目
- 需要逻辑判断显示的元素

> 💡 提示：如果某个元素无法选中，说明它是动态元素，需要通过提示词来修改。

## 编辑方式

### 直接编辑（免费）

选中元素后，可以直接修改以下属性：

#### 文本编辑
- 内容文字
- 字体大小
- 字体粗细
- 文字颜色

#### 颜色编辑
- 背景颜色
- 边框颜色
- 使用颜色选择器或输入 HEX 值

#### 布局调整
- 内边距 (padding)
- 外边距 (margin)
- 对齐方式

> 🎉 **免费特性**: 直接编辑不消耗积分！

### 提示词编辑（消耗积分）

选中元素后，也可以在输入框中输入提示词：

```
把这个按钮改成圆角，添加阴影效果
```

这种方式会消耗积分，但可以实现更复杂的修改。

## 使用场景

### 场景 1: 快速调整文案

选中标题 → 直接修改文字 → 点击保存

适合：调整页面文案、修正错别字

### 场景 2: 调整配色

选中元素 → 点击颜色选择器 → 选择新颜色

适合：更换主题色、调整视觉层次

### 场景 3: 微调布局

选中容器 → 调整 padding/margin 数值

适合：优化间距、改善视觉呼吸感

### 场景 4: 复杂样式变更

选中元素 → 输入提示词描述要求

适合：添加动画、改变布局方式、复杂视觉效果

## 最佳实践

### 1. 优先使用直接编辑

对于简单的文字、颜色、间距调整，直接编辑既快速又免费。

### 2. 组合使用

- 用提示词创建基础结构
- 用可视化编辑优化细节
- 用提示词添加复杂功能

### 3. 保存前预览

在可视化编辑模式下，修改是实时预览的，满意后再点击 **"Save"** 保存。

### 4. 批量修改用提示词

如果要修改多个相似元素，使用提示词更高效：

```
把所有的按钮都改成圆角，添加悬停效果
```

## 常见问题

### Q: 为什么有些元素选不中？

A: 该元素可能是动态生成的（如列表项、条件渲染的内容）。动态元素需要通过提示词修改。

### Q: 修改后能撤销吗？

A: 如果还没点击 "Save"，可以直接关闭编辑模式。已保存的修改可以通过版本历史回滚。

### Q: 可视化编辑会生成代码吗？

A: 会的，保存后 Lovable 会更新对应的代码文件。

### Q: 手机上能用可视化编辑吗？

A: 目前可视化编辑主要针对桌面端优化，移动端体验可能受限。

## 实用技巧

### 技巧 1: 快速改色

1. 启用编辑模式
2. 依次选中要改色的元素
3. 使用颜色选择器快速调整
4. 一次性保存所有更改

### 技巧 2: 对齐检查

使用可视化编辑可以清楚看到元素的边界，方便检查对齐和间距是否一致。

### 技巧 3: 设计微调

在设计基本完成后，使用可视化编辑进行最后的细节打磨，这个阶段通常不需要消耗积分。
